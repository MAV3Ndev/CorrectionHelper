# CorrectionHelper

 某予備校の採点業務を円滑にすることを目的として作成されたソフトウェア

## 機能

- グローバルスニペット
- 各大学の年度/大問ごとのスニペット

## 用法

[ここ](https://github.com/MAV3Ndev/CorrectionHelper/releases)から最新のバージョンを選択、.exeファイルをダウンロードしてインストール
あるいは自分でビルドしても良い

## 技術

- Nuxt3
- PrimeVue
- Electron
